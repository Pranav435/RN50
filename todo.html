ToDo App

In this project, you have to implement a Todo app. This is more of a website, but it should give you practice with React

Specification:

  • The user should be able to add todos, and delete them if they like
  • The user should be able to see these todos in a list, and the app should show a count with the number of todos the user has added

    • Getting started:

    Download the starter code:

    Here

    This should get you started, and should ensure that you only focus on the logic.

    What you have to do?

    The add-task method has already been created for you. You have to implement the delete task method, and the clear input method.

    Hints

    1. To clear input, use a blank string
    2. The tasks are an array, so you can just index into them.
